# Motion Director Style System

## Camera

- Use fast focal zooms on UI elements that matter.
- Zoom into the click target, hold briefly for legibility, then ease out.
- Keep screen orientation slightly tilted only during motion.
- Prefer beats as cut points. Use secondary beats for tiny scale pulses.

## Typography

- Start with black-screen typed text for dramatic hooks.
- Use one or two short lines.
- Side callouts should be labels, not paragraphs.
- Put callouts beside the active UI. Do not cover dense product text.

## Cursor

- Hide cursor except near clicks or important hovers.
- Use a visible ring expanding from the click target.
- Cursor motion should feel manual but quick.

## Product Proof

- Show real dashboard state.
- Show hardware/resources as capability-gated resources when relevant.
- Show MCP tools on screen, all at once when claiming tool count.
- Show metrics/proof near the end so the video closes with evidence.

## Anti-Patterns

- Do not overlay real screenshots onto an unrelated fake template.
- Do not use only right-left slide transitions.
- Do not repeat claims already made.
- Do not add decorative node graphics to every shot.
- Do not use AI-looking fake UI when real captures exist.
